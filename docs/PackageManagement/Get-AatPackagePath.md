# Get-AatPackagePath

## SYNOPSIS
Gets the package path for the working package.

## SYNTAX

```
Get-AatPackagePath [<CommonParameters>]
```

## DESCRIPTION
Gets the package path for the working package.

## EXAMPLES

### Example 1: Get the package path
```PowerShell
PS C:\> Get-AatPackagePath
C:\aat-package-examples\example-1
```

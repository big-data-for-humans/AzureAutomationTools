# Get-WorkingPackage

## SYNOPSIS
Gets the working package.

## SYNTAX

```
Get-WorkingPackage
```

## DESCRIPTION
Gets the working package, or if none has been set returns the first folder in
the working package folder (i.e.
as returned by Get-AatWorkingFolder)

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-AatWorkingPackage
```
